## 🚀 About This Repository

**Disclaimer:** This repository is a timeline of my JavaScript learning journey. It contains core programming concepts, DOM manipulation exercises, and basic mini-projects (like a functional Cart System) that I built to strengthen my logic and frontend skills.

It is meant to track my personal growth and serve as a quick reference guide for JavaScript fundamentals and ES6+ features.

### 📂 What's Inside:
* **Core Basics:** Variables, Data Types, Operators, and Conditional Statements.
* **Logic Building:** Loops, Arrays, Strings, and Functions.
* **DOM Manipulation:** Interacting with the webpage and handling user events.
* **Modern JS (ES6+):** Arrow functions, Destructuring, Map/Filter/Reduce.
* **Mini Projects:** Practical implementations including a Cart System.
